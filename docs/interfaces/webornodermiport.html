Interface WebOrNodeRmiPort

Minimal message port interface required by this module for remote method invocation (RMI).

Covers the Web API as well as the NodeJS API. Web Worker, ServiceWorker and MessagePort, as well as NoeJS Worker and parentPort are assignment compatible with this interface.
